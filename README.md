# Inventory Checker

This application is built to help a business check and manage its inventory.

## Local Development

1. Install dependencies:

```bash
pnpm install
```

2. Start the development server:

```bash
pnpm dev
```

3. Open the app in your browser:

```text
http://localhost:3000
```

Notes:
- If you use a different package manager, you can run `npm install && npm run dev` or `yarn install && yarn dev`.
